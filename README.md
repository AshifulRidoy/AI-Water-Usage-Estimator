# AI Water Usage Estimator

A browser extension that estimates and tracks the water footprint of AI prompts in real time. Supports ChatGPT, Claude, and Gemini.

## Features

- Real-time water usage estimates per AI response
- Inline UI below each response
- Dashboard with daily/lifetime totals
- Dark & light mode
- No backend, no tracking, no data collection

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Test
